Habilitar administrador de certificados de producción¶

Para agregar la aplicación «Administración de Certificados Digitales» a nuestro escritorio de AFIP primero debemos ingresar en «Administrador de Relaciones de Clave Fiscal»

En caso de tener permiso para administrar las relaciones de otros contribuyentes nos hará elegir el de quien queremos administrar, caso contrario se elegirá el nuestro automáticamente.

Luego elegimos «Adherir servicio»

Allí dentro elegimos «AFIP > Servicios interactivos > Administración de Certificados Digitales»

Le damos a confirmar y ya tendremos acceso a la aplicación desde nuestro escritorio de Afip
